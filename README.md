# Mini-chat-bot
This project is an AI-powered chatbot that uses Retrieval-Augmented Generation (RAG) and Ollama to generate accurate, context-aware responses from a custom knowledge base. The AI backend is implemented in Python, while the web application is built with Node.js, JavaScript, and EJS. Communication between the frontend and backend is handled through REST APIs, providing a seamless and interactive user experience.

### Tech Stack
- **AI/LLM:** Ollama, RAG
- **Backend:** Python, Node.js
- **Frontend:** JavaScript, EJS
- **Communication:** REST API
